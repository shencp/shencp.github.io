● 5 执行**autoconf | with sudo **

├── configure.ac
├── main.c

● 6 执行automake –add-missing | with sudo

**├── aclocal.m4
├── autom4te.cache
│   ├── output.0
│   ├── output.1
│   ├── requests
│   ├── traces.0
│   └── traces.1
├── autoscan.log
├── compile -> /usr/share/automake-1.16/compile
├── config.h.in
├── configure
├── configure.ac
├── depcomp -> /usr/share/automake-1.16/depcomp
├── install-sh -> /usr/share/automake-1.16/install-sh
├── main.c
├── Makefile.am
├── Makefile.in
└── missing -> /usr/share/automake-1.16/missing

经过以上操作我们就可以像一般安装程序一config –> make –> make install
